# Hotel Reservation Platform

## Project Overview
This is a full-stack hotel reservation web application that allows users to search, explore, and book hotels based on their preferences. It also enables hotel owners to list their properties and manage bookings.

The platform is designed to simplify the booking experience while providing a seamless interface for both customers and hotel providers.

---

## Features

- User Authentication (Login and Registration)
- Hotel Listing and Management for Owners
- Search and Filter Hotels (price, location, amenities)
- Secure Booking and Payment Integration
- Image Upload Support for Hotels
- Interactive and User-Friendly Interface

---

## Tech Stack

**Frontend:**
- React.js
- HTML, CSS, JavaScript

**Backend:**
- Node.js
- Express.js

**Database:**
- MongoDB

**Other Tools:**
- Cloudinary (Image Upload)
- Stripe (Payment Integration)
